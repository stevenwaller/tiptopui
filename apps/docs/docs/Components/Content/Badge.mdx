## Appearance

```jsx live noInline
const appearances = ['solid', 'faded', 'outline', 'ghost', 'text'];

render(
  <div style={{display: 'flex', alignItems: 'center', gap: '10px'}}>
    {appearances.map(appearance => (
      <Badge key={appearance} appearance={appearance}>
        {appearance}
      </Badge>
    ))}
  </div>
);
```

## Empty Badge

When no content is provided, the badge renders as just a shape with color.

```jsx live
<div style={{display: 'flex', alignItems: 'center', gap: '10px'}}>
  <Badge />
  <Badge variant="danger" />
  <Badge variant="success" />
  <Badge variant="warning" />
</div>
```

## Slots

```jsx live

  <Badge slotStart={<IconStar size={14}/>}>Starred</Badge>
  <Badge slotEnd={<IconX size={14}/>}>Close</Badge>
```

## Show Outline

The `showOutline` prop adds a white outline to help the badge stand out over backgrounds.

```jsx live
<div style={{background: 'linear-gradient(45deg, #ff6b6b, #4ecdc4)', padding: '20px', borderRadius: '8px'}}>
  <div style={{display: 'flex', gap: '10px', alignItems: 'center'}}>
    <Badge>No Outline</Badge>
    <Badge showOutline>With Outline</Badge>
    <Badge variant="danger" showOutline>99+</Badge>
    <Badge variant="success" showOutline shape="circle">âœ“</Badge>
  </div>
</div>
```

## Positioning

Badges can be absolutely positioned relative to their parent container using the `position` prop.

```jsx live
<div style={{display: 'flex', gap: '30px', alignItems: 'center'}}>
  <div style={{position: 'relative', width: '60px', height: '60px', background: '#f0f0f0', borderRadius: '8px'}}>
    <Badge position="top-right" variant="danger">5</Badge>
  </div>

  <div style={{position: 'relative', width: '60px', height: '60px', background: '#f0f0f0', borderRadius: '8px'}}>
    <Badge position="top-left" variant="success" shape="circle" />
  </div>

  <div style={{position: 'relative', width: '60px', height: '60px', background: '#f0f0f0', borderRadius: '8px'}}>
    <Badge position="bottom-right" variant="warning">New</Badge>
  </div>

  <div style={{position: 'relative', width: '60px', height: '60px', background: '#f0f0f0', borderRadius: '8px'}}>
    <Badge position="bottom" variant="neutral">Info</Badge>
  </div>
</div>
```
