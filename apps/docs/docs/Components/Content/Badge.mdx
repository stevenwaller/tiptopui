## Appearance

```jsx live noInline
const appearances = ["solid", "faded", "outline", "ghost", "text"];

render(
  <div style={{ display: "flex", alignItems: "center", gap: "10px" }}>
    {appearances.map((appearance) => (
      <Badge key={appearance} appearance={appearance}>
        {appearance}
      </Badge>
    ))}
  </div>
);
```

## Variant

```jsx live noInline
const variants = [

];
```

## Empty Badge

When no content is provided, the badge renders as just a shape with color.

```jsx live
<div style={{ display: "flex", alignItems: "center", gap: "10px" }}>
  <Badge />
  <Badge variant="danger" />
  <Badge variant="success" />
  <Badge variant="warning" />
</div>
```

## Slots

Content can be added to the start and end of the badge using the `slotStart` and `slotEnd` props.

```jsx live
// Icons from https://tabler.io/docs/icons/react
<div style={{ display: "flex", gap: "10px" }}>
  <Badge slotEnd={<IconArrowRight size={20} />}>With icon</Badge>
  <Badge slotStart="ðŸ˜€">With Emoji</Badge>
  <Badge slotStart="'Slot Start'" slotEnd="'Slot End'">
    With Text
  </Badge>
</div>
```

## Fixed Ratio

When just using icons, you can set the `fixedRatio` prop to make sure the badge is always a square.

```jsx live
<div style={{ display: "flex", gap: "10px", alignItems: "center" }}>
  <Badge shape="rounded" fixedRatio>
    <IconBell size={14} />
  </Badge>
  <Badge shape="pill" variant="success" fixedRatio>
    <IconCheck size={14} />
  </Badge>
  <Badge shape="circle" variant="danger" fixedRatio>
    <IconAlertTriangle size={14} />
  </Badge>
  <Badge shape="square" variant="warning" fixedRatio>
    <IconClock size={14} />
  </Badge>
</div>
```

## Show Outline


## Positioning

Badges can be absolutely positioned relative to their parent container using the `position` prop.

```jsx live
<div style={{ display: "flex", gap: "30px", alignItems: "center" }}>
  <div
    style={{
      position: "relative",
      width: "60px",
      height: "60px",
      background: "#f0f0f0",
      borderRadius: "8px",
    }}
  >
    <Badge position="top-right" variant="danger">
      5
    </Badge>
  </div>

  <div
    style={{
      position: "relative",
      width: "60px",
      height: "60px",
      background: "#f0f0f0",
      borderRadius: "8px",
    }}
  >
    <Badge position="top-left" variant="success" shape="circle" />
  </div>

  <div
    style={{
      position: "relative",
      width: "60px",
      height: "60px",
      background: "#f0f0f0",
      borderRadius: "8px",
    }}
  >
    <Badge position="bottom-right" variant="warning">
      New
    </Badge>
  </div>

  <div
    style={{
      position: "relative",
      width: "60px",
      height: "60px",
      background: "#f0f0f0",
      borderRadius: "8px",
    }}
  >
    <Badge position="bottom" variant="neutral">
      Info
    </Badge>
  </div>
</div>
```

### Interactive Badges with Close

```jsx live
<div style={{ display: "flex", gap: "8px", flexWrap: "wrap" }}>
  <Badge

        size={14}

        onClick={() => alert("Remove tag")}
      />
    }
    variant="primary"
  >
    Tag 1
  </Badge>
  <Badge
    slotEnd={
      <IconX
        size={14}
        style={{ cursor: "pointer" }}
        onClick={() => alert("Remove tag")}
      />
    }
    variant="secondary"
  >
    Tag 2
  </Badge>
  <Badge
    slotEnd={
      <IconX
        size={14}
        style={{ cursor: "pointer" }}
        onClick={() => alert("Remove tag")}
      />
    }
    variant="success"
  >
    Tag 3
  </Badge>
</div>
```
